# Responsive Landing Page

A modern and responsive landing page built using **HTML5** and **CSS3**. The project focuses on creating a clean, responsive, and user-friendly interface inspired by modern business websites.

## Features

- Responsive Navigation Bar
- Hero Section with Call-to-Action
- Modern UI Design
- Responsive Layout using Flexbox
- Custom Buttons
- Floating Information Card
- Student Avatars Section
- Google Fonts Integration
- Font Awesome Icons
- Hover Effects and Smooth Transitions
- Clean and Organized Code

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome
